Review the pasted or attached message and improve it using the Nonviolent Communication (NVC) framework.

Use this structure:

1. **Observation**: State what happened without blame or judgment.
2. **Feeling**: Express the speaker's feeling clearly.
3. **Need**: Identify the underlying need or value.
4. **Request**: End with a clear, respectful, actionable request.

Keep the same intent as the original message, but make it calmer, clearer, and more constructive.

Return:
- **Improved message**
- **Brief breakdown** of Observation, Feeling, Need, and Request used
